# 📈 Stock Price Prediction System

## 📌 Overview

This project is a Multi-Company Stock Price Prediction System developed using Machine Learning and Deep Learning techniques.

The system analyzes historical stock market data and predicts future stock prices using the LSTM (Long Short-Term Memory) Deep Learning model.

The project also includes interactive financial visualizations and a Streamlit dashboard for real-time user interaction.

---

# 🚀 Features

✅ Multi-Company Stock Selection  
✅ Historical Stock Price Analysis  
✅ Interactive Candlestick Chart  
✅ Moving Average Analysis (50-Day & 200-Day)  
✅ LSTM Deep Learning Prediction  
✅ Actual vs Predicted Price Visualization  
✅ RMSE & MAE Accuracy Metrics  
✅ Future 7-Day Stock Forecasting  
✅ Streamlit Interactive Dashboard

# 🏗️ Model Used

## LSTM (Long Short-Term Memory)

LSTM is a Deep Learning Recurrent Neural Network (RNN) architecture specially designed for Time Series Forecasting problems.

It helps in learning stock market trends and predicting future prices based on historical patterns.
